# Installation

## Bootstrapper

The bootstrapper depends on three things: git, ruby and rake.

To install, just run:

```bash
zsh < <( curl https://raw.github.com/mmozuras/dotfiles/master/bootstrap.sh )
```

## Stay Updated

To get updated, just run the bootstrapper again:

```bash
sh ~/.dotfiles/bootstrap.sh
```
