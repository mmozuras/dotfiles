# Installation

## Bootstrapper

The bootstrapper depends on three things: ruby, rake, and bundler.

To install, just run:

    zsh < <( curl https://raw.github.com/gf3/dotfiles/master/bootstrap.sh )

## Stay Updated

To get updated, just run the bootstrapper again:

    ~/.dotfiles/bootstrap.sh
